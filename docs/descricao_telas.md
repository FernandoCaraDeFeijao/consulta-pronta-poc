# Descrição das telas e funcionalidades

- Autenticação: Login e cadastro com diferenciação entre paciente e profissional.

- Perfil: Visualização de todas as informações do próprio perfil

- Edição de perfil: Formulário para a edição de todas as informações do próprio perfil

- Paciente:
	- Dashboard: Painel com resumo dos últimos sintomas, mostrando prescrições e exames.
	- Sintomas: Visualização de cada sintoma e seus detalhes.
	- Registro de sintomas: Formulário para registrar intensidade (1-10), local do corpo, descrição, data e horário.
	- Relatórios: Visualização do relatório simples de evolução e comparação de melhora/piora dos sintomas, junto das anotações do médico.
	- Geração de relatório: Formulário para gerar relatórios com base em um certo periodo
	- Profissionais: Visualização de profissionais de saúde para poder editar suas autorizações

- Profissional de Saúde
	- Dashboard: Painel com seus pacientes
	- Prontuário: Visualização do prontuário do paciente (dados de saúde, relatórios compartilhados, prescrições, exames), junto da opção de fazer anotações
	- Prescrição: Formulário para gerar prescrição de um paciente
	- Exame: Formulário para solicitar exame de um paciente
